
#SumOFListOfNumbers API using Flask
http://localhost:5000/total

##### Install dependencies
```bash
pip install Flask
```
## Install guide

##### Clone the repo

```bash
$ git clone https://github.com/ktchandran/SumOFListOfNumbers.git
$ cd SumOFListOfNumbers
```

##### Run the app
```bash
$ python run_app.py
```


## Test

```bash
$ python tests.py
```
